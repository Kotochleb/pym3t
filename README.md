A python wrapper around ICG tracker, __WIP__

Building
----

`git clone git@github.com:MedericFourmy/pyicg.git --recursive`

Install dependencies:  
`conda create -n pyicg`  
`conda activate pyicg`  
`mamba update --file environment.yaml`

Then  
`pip install .`
